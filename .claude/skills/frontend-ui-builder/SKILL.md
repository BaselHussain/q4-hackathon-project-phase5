# Frontend UI Development

## Instructions

1. **Page structure**
   - Semantic HTML layout
   - Header, main, footer separation
   - Reusable sections and components

2. **Components**
   - Buttons, cards, forms, navbars
   - Reusable and configurable props
   - Consistent spacing and typography

3. **Layout & Styling**
   - Flexbox and Grid for layout
   - Responsive breakpoints
   - Modern styling with Tailwind, CSS, or component libraries

4. **Responsiveness**
   - Mobile-first design
   - Adaptive layouts for tablet and desktop
   - Touch-friendly UI elements

## Best Practices
- Keep components small and reusable
- Follow a consistent design system
- Use semantic HTML for accessibility
- Avoid inline styles
- Optimize for performance and readability

## Example Structure
```html
<main class="container">
  <header class="page-header">
    <h1 class="title">Page Title</h1>
  </header>

  <section class="grid-layout">
    <div class="card">
      <h2>Card Title</h2>
      <p>Card content goes here.</p>
      <button class="primary-btn">Action</button>
    </div>
  </section>
</main>
